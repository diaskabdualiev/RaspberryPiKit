Первый запуск
==================

Этот раздел содержит подробные инструкции для начала работы с вашим набором Raspberry Pi 5. 
Здесь вы найдете пошаговые руководства от первоначальной сборки устройства до запуска 
вашего первого кода. Раздел структурирован в логической последовательности действий, 
которые необходимо выполнить для успешного старта с Raspberry Pi.

Следуя этим инструкциям, вы сможете правильно подключить все компоненты, включить 
устройство, настроить операционную систему и запустить свою первую программу. Это 
заложит прочную основу для дальнейшего изучения материалов курса.
